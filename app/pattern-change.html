This is great for composing some things from scratch but all the sudden you're repeating yourself, but you still want to use some immutable styles. Wat do?

Composing components

Not part of SASS or traditional CSS, through the magic of postCSS we're able to cobble together the functional css classes into more tradtional classes via the postCSS plugin apply-class.

                        
.card {
    @apply .bd(1), .bdrad(3), .bdc(gray-l1, .p(2), .d(f), .ai(t);
}

Which then is postCSS'd into:

                        
.card {
  border-style: solid;
  border-width: 1px;
  border-radius: 1rem;
  padding: 0.5rem;
  display: flex; }

Why not just write this CSS in the first place? When you consume the same composable classes and combine them into a component, you're just consuming all the patterns in a functional way and any changes to overall design goals like spacing or font weight changes are just automatically consumed.
